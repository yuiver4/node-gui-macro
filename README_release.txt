========================================================
 ImgMacro - 이미지/텍스트 인식 자동 클릭 매크로
========================================================

이 폴더를 통째로 원하는 PC에 복사하면 됩니다.
(파이썬/Tesseract 설치 불필요. 한글 OCR 엔진 내장)

★ Editor.exe 를 실행하세요 ★  ← 이것만 열면 됩니다.

--------------------------------------------------------
폴더 구성
--------------------------------------------------------
  Editor.exe     ← 매크로를 만드는 GUI (여기서 다 함)
  ImgMacro.exe   매크로 실행 엔진 (에디터가 자동 호출)
  Snip.exe       화면 캡처/한글 입력 도구 (에디터가 자동 호출)
  tesseract\     한글 OCR 엔진 (건드리지 마세요)
  templates\     캡처한 이미지가 저장되는 곳

--------------------------------------------------------
사용법
--------------------------------------------------------
1) Editor.exe 실행

2) 노드 추가
   - 빈 곳에서 "마우스 우클릭" -> 추가 메뉴  (또는 상단 [추가] 메뉴)
   - 페이즈 / 반복(횟수) / 딜레이 / 종료 중 선택

3) 페이즈 노드 설정
   - 방식: 이미지 / 텍스트
   - 이미지: [캡처] 누르고 화면에서 버튼을 드래그
   - 텍스트: 찾을 문구 입력
   - ※ 한글 입력칸에 한글이 안 써지면 옆의 [한글] 버튼을 눌러 입력하세요
     (입력칸 직접 타이핑은 한글이 깨질 수 있음)

4) 노드 연결 (Shader Graph 처럼)
   - [시작]의 '시작 >' 을 첫 노드의 '이전' 으로 드래그
   - 페이즈의 '성공 >' -> 다음 노드 / '실패 >' -> 실패 시 갈 노드
   - 반복: '반복 >' = 반복할 본문(끝나면 이 노드로 되돌아오게 연결),
           '완료 >' = 지정 횟수를 다 채운 뒤 진행할 곳
   - 딜레이: '다음 >' 로 이어서 지정 초만큼 대기

5) [파일] -> [저장] (.json)

6) 상단 [▶ 실행] 클릭 (3초 뒤 시작, 그 사이 대상 프로그램을 띄워두세요)
   - 실행 중 "현재 진행 노드"가 노란색으로 표시됩니다
   - 검은 로그 창에 진행 상황이 출력됩니다
   * [미리보기(클릭 없음)] 로 클릭 없이 인식만 먼저 확인 가능

--------------------------------------------------------
대상 창 지정 (전체화면 대신 특정 창만)
--------------------------------------------------------
  [전역 설정] -> "캡처 대상"을 [특정 창] 으로 -> [창 선택...] 에서 프로그램 선택
  - 그 창의 화면만 인식/클릭하므로 더 가볍고 정확합니다.
  - 창이 움직여도 따라갑니다. (전체화면 모드도 그대로 사용 가능)

--------------------------------------------------------
키매핑 (단축키 -> 클릭)  ※ 매크로와 별개 기능
--------------------------------------------------------
  단축키 없는 게임에서, 키를 누르면 지정 위치를 대신 클릭해 줍니다.
  1) [전역 설정]에서 대상 창을 지정
  2) [키매핑] 창 -> [+ 행 추가] -> 키 입력(q, f1, space 등) ->
     [위치 지정] 누르고 화면에서 클릭할 지점을 한 번 클릭
     (창 크기에 대한 '비율'로 저장되어 해상도가 달라도 같은 위치를 누름)
  3) [키매핑 시작] -> 이제 그 키를 누르면 대상 창의 그 위치가 클릭됨
     (정지 키 F12 로 종료)

--------------------------------------------------------
노드 종류 / 페이즈 옵션
--------------------------------------------------------
  - 페이즈: 이미지/텍스트 인식 -> 클릭. 옵션:
      클릭함(끄면 '감지만' = 분기에만 사용, 클릭 안 함),
      사라짐확인, 제한(초), 간격(0=기본, 예: 10 -> 10초마다 검사하며 대기),
      재클릭, 랜덤±(클릭 위치를 ±N px 무작위로)
  - 마우스 이동: 대상(창) 중앙으로 마우스 이동
  - 반복(횟수): 구간을 N번 반복
  - 딜레이: 고정/랜덤(최대>0이면 둘 사이 랜덤) 대기
  - 이미지가 없으면 노드에 빨간 "이미지 없음" 표시 -> [캡처]로 채우세요

--------------------------------------------------------
편의 기능
--------------------------------------------------------
  - 마우스 휠: 확대 / 축소        - 우클릭: 노드 추가 메뉴
  - [한글] 버튼: 한글 입력창       - Del: 선택 삭제
  - 저장/열기: 윈도우 기본 파일 창
  - [전역 설정]: 캡처 대상(창/전체화면), 정확도, 정지키, "실행 시 최소화" 등

--------------------------------------------------------
예제 흐름
--------------------------------------------------------
  마비노기_던전반복.json  - [파일]->[열기]로 불러와 구조 확인.
  (이미지 노드들이 "이미지 없음" 빨강으로 뜨면 각 노드 [캡처]로 채우세요)

※ 실행 중 에디터가 켜져 있으면 대상 화면을 가릴 수 있습니다.
  에디터는 대상 프로그램과 "다른 모니터"에 두세요.
  (전역 설정에서 "실행 시 에디터 최소화" 를 켜면 자동으로 내려갑니다)

--------------------------------------------------------
비상 정지 (언제든 멈추기)
--------------------------------------------------------
  - 마우스를 화면 왼쪽 위 끝(0,0)으로 휙 이동
  - 또는 F12 키
  - 또는 로그 창에서 Ctrl+C

--------------------------------------------------------
잘 안 될 때
--------------------------------------------------------
  - 인식이 약하면: 노드의 '정확도'를 조금 낮추거나, [영역]으로 범위를 좁히세요.
  - 글자 인식: 큰 화면(4K)은 [영역] 사용을 권장합니다.
  - 다른 배율(125%/150%) PC에서도 이미지가 인식되도록 멀티스케일이 켜져 있습니다.
    그래도 안 잡히면 그 PC에서 다시 캡처하세요.
