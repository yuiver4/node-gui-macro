========================================================
 ImgMacro - 이미지/텍스트 인식 자동 클릭 매크로
========================================================

이 폴더를 통째로 원하는 PC에 복사하면 됩니다.
(파이썬/Tesseract 설치 불필요. 한글 OCR 엔진 내장)

★ Editor.exe 를 실행하세요 ★  ← 이것만 열면 됩니다.

--------------------------------------------------------
폴더 구성
--------------------------------------------------------
  Editor.exe     ← 매크로를 만드는 GUI (여기서 다 함)
  ImgMacro.exe   매크로 실행 엔진 (에디터가 자동 호출)
  Snip.exe       화면 캡처 도구 (에디터가 자동 호출)
  tesseract\     한글 OCR 엔진 (건드리지 마세요)
  templates\     캡처한 이미지가 저장되는 곳

--------------------------------------------------------
사용법 (GUI)
--------------------------------------------------------
1) Editor.exe 실행

2) 상단 메뉴 [추가] -> [＋ 페이즈 노드] 로 노드 추가

3) 노드에서:
   - '방식'을 이미지/텍스트 중 선택
   - 이미지: [📷 캡처] 누르고 화면에서 버튼을 드래그
   - 텍스트: 찾을 문구 입력 (예: 확인)

4) 노드들을 선으로 연결 (Shader Graph 처럼)
   - [시작]의 '시작 ▶' 을 첫 페이즈의 '◀ 이전' 으로 드래그
   - 페이즈의 '성공 ▶' -> 다음 페이즈로
   - 페이즈의 '실패 ▶' -> 실패 시 갈 노드로 (다른 페이즈/종료 등)
   - [추가] -> [＋ 종료 노드] 로 성공/실패 종료 지점 배치

5) [파일] -> [저장] 으로 흐름 저장 (.json)

6) 상단 [▶ 실행] 클릭
   - 3초 뒤 시작되니 그 사이 대상 프로그램을 띄워두세요
   - 에디터는 자동으로 최소화되고, 검은 로그 창에 진행이 표시됩니다

* [미리보기(클릭없음)] 로 실제 클릭 없이 인식만 먼저 확인해 볼 수 있습니다.

--------------------------------------------------------
흐름이 어떻게 동작하나
--------------------------------------------------------
각 페이즈: 대상 인식 -> 클릭 -> (옵션) 사라졌는지 확인
  성공하면 '성공 ▶' 선을 따라 다음 노드로
  실패하면(시간초과 등) '실패 ▶' 선을 따라 지정한 노드로
종료 노드에 도달하면 매크로가 끝납니다.

각 노드에서 정할 수 있는 것:
  정확도, 클릭 후 사라짐 확인 여부, 제한시간(초), 재클릭 횟수,
  🔲 영역(검색 범위를 좁혀 빠르고 정확하게 — 큰 화면에서 권장)

--------------------------------------------------------
비상 정지 (언제든 멈추기)
--------------------------------------------------------
  - 마우스를 화면 왼쪽 위 끝(0,0)으로 휙 이동
  - 또는 F12 키
  - 또는 로그 창에서 Ctrl+C

--------------------------------------------------------
잘 안 될 때
--------------------------------------------------------
  - 인식이 약하면: 노드의 '정확도'를 조금 낮추거나, 🔲 영역으로 범위를 좁히세요.
  - 글자 인식: 큰 화면(4K)은 🔲 영역 사용을 권장합니다.
  - 다른 배율(125%/150%) PC에서도 이미지가 인식되도록 멀티스케일이 켜져 있습니다.
    그래도 안 잡히면 그 PC에서 다시 캡처하세요.
  - [전역 설정] 에서 모니터 번호, 기본 정확도, 정지키 등을 바꿀 수 있습니다.
